---
title: "Get Swap Data"
description: "Get transaction data needed to execute a token swap"
---

## Method

```typescript
async getSwapData(params: SwapParams): Promise<SwapResponseData>
```

Retrieves the complete transaction data needed to execute a swap, including routing information and transaction parameters.

## Parameters

<ParamField path="params" type="SwapParams" required>
  Swap request parameters

  <Expandable title="properties">
    <ParamField path="chainIndex" type="string">
      Chain identifier (e.g., "1" for Ethereum, "501" for Solana)
    </ParamField>

    <ParamField path="fromTokenAddress" type="string" required>
      Contract address of the token to swap from
    </ParamField>

    <ParamField path="toTokenAddress" type="string" required>
      Contract address of the token to swap to
    </ParamField>

    <ParamField path="amount" type="string" required>
      Amount to swap (in token's smallest unit)
    </ParamField>

    <ParamField path="slippagePercent" type="string">
      Maximum acceptable slippage (0-1, e.g., "0.005" for 0.5%). Required if autoSlippage is false.
    </ParamField>

    <ParamField path="autoSlippage" type="boolean">
      Enable automatic slippage calculation
    </ParamField>

    <ParamField path="maxAutoSlippagePercent" type="string">
      Maximum slippage when autoSlippage is enabled (required with autoSlippage)
    </ParamField>

    <ParamField path="userWalletAddress" type="string">

    <ParamField path="swapReceiverAddress" type="string">
      Address to receive swapped tokens (defaults to userWalletAddress)

    <ParamField path="directRoute" type="boolean">

    </ParamField>

    <ParamField path="toTokenReferrerWalletAddress" type="string">
      Referrer wallet address for destination token fees
    </ParamField>

    <ParamField path="positiveSlippagePercent" type="string">
      Positive slippage percentage
    </ParamField>

    <ParamField path="gasLimit" type="string">
      Gas limit for the transaction (EVM chains)
    </ParamField>

    <ParamField path="gasLevel" type="string">
      Gas level: "low", "medium", or "high" (EVM chains)
    </ParamField>

    <ParamField path="computeUnitPrice" type="string">
      Compute unit price (Solana only)
    </ParamField>

    <ParamField path="computeUnitLimit" type="string">
      Compute unit limit (Solana only)
    </ParamField>

    <ParamField path="callDataMemo" type="string">

    </ParamField>
  </Expandable>
</ParamField>

## Example

```typescript
import { OKXClient } from '@okxweb3/okx-api';

const client = new OKXClient({
  apiKey: 'your-api-key',
  secretKey: 'your-secret-key',
  apiPassphrase: 'your-passphrase',
  projectId: 'your-project-id'
});

const swapData = await client.dex.getSwapData({
  chainIndex: '1',
  fromTokenAddress: '0xEeeeeEeeeEeEeeEeEeEeeEEEeeeeEeeeeeeeEEeE',
  toTokenAddress: '0xA0b86991c6218b36c1d19D4a2e9Eb0cE3606eB48',
  amount: '1000000000000000000', // 1 ETH
  slippagePercent: '0.005',
  userWalletAddress: '0x742d35Cc6634C0532925a3b844Bc454e4438f44e'
});

console.log('Transaction data:', swapData.data[0].tx);
console.log('Router result:', swapData.data[0].routerResult);
```

## Validation

The method validates:
- Either `slippagePercent` or `autoSlippage` must be provided
- `slippagePercent` must be between 0 and 1
- `maxAutoSlippagePercent` is required when `autoSlippage` is enabled
